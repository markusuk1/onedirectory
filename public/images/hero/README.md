# Regional Hero Images

Drop one realistic hero background image per region in this folder.

Required filenames:

- `northeast.jpg` (e.g. Angel of the North / Newcastle skyline)
- `northwest.jpg` (e.g. Manchester/Liverpool iconic skyline)
- `scotland.jpg` (e.g. Edinburgh/Glasgow iconic scene)
- `midlands.jpg` (e.g. Birmingham skyline)
- `yorkshire.jpg` (e.g. York/Leeds iconic scene)
- `east.jpg` (e.g. Cambridge/Norwich iconic scene)
- `london.jpg` (e.g. Big Ben / Westminster)
- `southeast.jpg` (e.g. Brighton Pier / White Cliffs)
- `southwest.jpg` (e.g. Clifton Suspension Bridge / Cornwall coast)
- `wales.jpg` (e.g. Cardiff Bay / Welsh landmark)

Recommended:
- Landscape ratio (roughly 16:9)
- Width around 2000px
- Keep visuals clear at low opacity
- No text burned into images

These are shown behind the homepage hero banner with an overlay.
